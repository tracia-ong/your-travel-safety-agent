# 🛰 Vanguard Scout: Travel Safety Swarm
An AI-agent swarm built with ADK to synthesize real-time travel safety intelligence from unstructured data.

## 📋 Project Overview
Vanguard Scout is an autonomous travel intelligence swarm built with ADK to synthesize real-time travel safety intelligence from unstructured data.

## ⚙️ System Architecture (A2A Flow)
Our system utilizes an Agent-to-Agent (A2A) flow:
1. Intel Scout scans unstructured news and government advisories.
2. Data is passed to the Safety Strategist.
3. Safety Strategist reasons over the data and provides a risk verdict.

## 🤖 Agent Profiles
* Intel Scout: Responsible for data retrieval and discovery of safety alerts.
* Safety Strategist: Responsible for reasoning, risk analysis, and applying safety guardrails.

## 🚀 Setup Instructions
1. Clone the repo.
2. Install requirements: pip install -r requirements.txt
3. Set up .env with your GEMINI_API_KEY.
4. Run the swarm: python3 agents/scout.py
